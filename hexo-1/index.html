记hexo+next搭建个人博客

发表于 2019-01-14 | 本文总阅读量# 次
字数统计: 1.1k 字 | 阅读时长 ≈ 4 分钟

前言

前段时间腾讯云买的服务器到期了，偶然看到GitHub page搭建的网站，感觉好炫酷，然后自己也尝试搭建一个，看到别人用了两天时间，自己却断断续续花了一周时间，碰到了各种奇葩的bug，顺手记录一下吧！！！

本地服务

环境配置

  1. Node.js

    Node.js安装教程

  2. Git

    Git安装教程

  3. 注册Github账号

    https://github.com ~ (全球最大同性交友平台)~

安装Hexo

node.js和git安装完成后，鼠标右键点击“git bash here”进入Git shell安装hexo

npm install -g hexo-cli

Start

在电脑任意位置，新建一个文件夹（如：G:\hexo），进入文件夹，鼠标右击鼠标右键点击“git bash here”执行命令

hexo init
npm install

开启本地服务

hexo s

服务开启后，在浏览器中输入 http://localhost:4000/
如果4000端口没有被占用的话，浏览器应该显示hexo自带的一篇hello-world
hexo成功安装！

新建文章

打开Git shell，Ctrl+C停止本地服务，执行命令
> hexo new post "文件名"
打开hexo文件夹根目录，G:\hexo\source_posts 文件夹下 文件名.md文件，遵循markdown语法，开始你的第一篇博客吧
打开Git shell，执行命令

hexo g
hexo s

打开浏览器，输入http://localhost:4000查看你的第一篇博客吧！

Github

注册Github账号

打开浏览器，输入https://github.com

英语二级的同学Google可以翻译一下

新建仓库

GitHub邮箱认证后才可以创建仓库，注册完成后，打开自己邮箱认证一下


仓库名字 你的GitHub名称.github.io
我的GitHub账户名：Southwindaa
我的GitHub仓库名：Southwindaa.github.io

然后public，create下一步，

配置SSH key

打开git shell

  • 设置username和email

在你的电脑中找到C:\Users\你的账户名.ssh \id_rsa.pub文件，复制内容到GitHub里

部署

打开hexo文件夹根目录，打开配置文件（ _config.yml ），在文件末尾位置修改如下：

git clone git@github.com:iissnan/hexo-theme-next.git

theme: next
`

注意yml文件格式theme：后面有个空格

hexo s开启本地服务，查看自己的next主题吧！

hexo clean
hexo g
hexo d

显示成功后，上传到你的GitHub吧！打开浏览器输入https://你的github用户名.github.io浏览你的网页吧

更换域名

如果你感觉域名太长了，不够炫酷，也可以尝试用自己域名！

提示：你的用户名.github.io也可以访问网站，并不一定需要购买域名。例：Southwindaa.github.io和www.lwxblog.club都可以进入网站

域名注册

域名注册可以选择阿里云，华为云，腾讯云，也可以选择freedom（免费的）
我用的是腾讯云（懒得注册和身份认证了）

首年便宜，续费贵；club便宜，com贵，所以遇到心动域名买个十年，买到就是赚到！！！

加入购物车，结账~

完成身份认证后，找到自己购买的域名，点击解析。@是为了不输入www，直接lwxblog.club就可以访问，记录值需要自己ping一下自己的github然后输入自己ping到的ip，win+R，输入：ping 你的GitHub账户名.github.io

完成后需要在Source下新建一个CANME文件（文件没有后缀！），输入你购买的域名（如：www.lwxblog.club），然后重新上传一下，打开GitHub网站，找到你的仓库，找到setting，找到GitHub page，在Custom domain中输入拟购买的域名，完成后save

结尾

自此网站搭建完成，打开浏览器，输入你购买的域名，你将有1%的可能看到你的hexo+next博客（忘得差不多了~），不试试你怎么知道你是不是1%！（手动狗头）
